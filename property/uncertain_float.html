Uncertain Float ( antares.model.property.UncertainFloat )¶

class antares.model.property.UncertainFloat¶

Bases: object

Represents a triple floats which is one of the data type of antares.attribute.Property. It consists of three float values, one being the expected value, a second being the lower std dev (one std dev below) and the third being the upper std dev (one std dev above).
