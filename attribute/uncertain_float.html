Uncertain Float ( antares.attribute.UncertainFloat )¶

class antares.attribute.UncertainFloat¶

Represents a triple floats which is one of the data type of antares.attribute.Attribute. It consists of three float values, one being the expected value, a second being the lower std dev (one std dev below) and the third being the upper std dev (one std dev above).

Attributes Summary

expected_value The first float: expected value.
lower_stddev The second float: lower std dev (one std dev below).
upper_stddev The third float: upper std dev (one std dev above).
